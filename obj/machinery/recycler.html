Space Station 13 - Nebula13 - Modules - Types — Var Details - Proc Details

recycler
/obj/machinery/recycler

Recycling machine for crushing trash and sometimes mobs into raw materials.

Vars

crush_listenerListener for things coming into the machine
efficiencyPercentage of materials recovered from the recycled item. Starting value is the maximum it'll go
emergency_stop_timeAmount of time the recycler will pause when it encounters an object triggering its emergency stop mode.
emergency_stopping_objectsThings that caused the machine to emergency stop. When it's empty, we're clear to restart, if it has anything in it, we still have something blocking us
icon_prefixPrefix of the icon state for the grinder
max_efficiencyThe maximum possible efficiency value for this machine.
recently_dispensed_matsList of the mats we've just dispensed, to avoid processing them as trash.
recycler_blacklistTypes that should never be allowed in the recycler
recycler_safety_blacklistTypes of objects which are ignored only when the safeties are on
recycler_stateBitflag for keeping track of internal status
sound_all_clearSound played when an obstruction is cleared and the machine is restarting
sound_emergencySound played when something forbidden cause an emergency stop
sound_idleLooped sound of the machine idling
sound_loopingSound token to the looping grinder sound
sound_obstructedSound played when something is still blocking the machine into emergency stop mode
sound_outputsSound played when materials are released from the machine
sound_processingSound played when something is being grinded by the machine
stored_materialsThe solid materials that were reclaimed from the recycled atoms. Kept separately from reagents, since phase shift isn't really working well rn.
time_last_cleared_msgTime we last told users an item clogging us was cleared
time_last_crushTime we last crushed something
time_last_crush_soundTime we last emitted a crush sound
time_last_outputTime we last outputed material dust
time_last_warning_msgTime we last told users something bad is happening
time_last_warning_soundTime we last issued a warning tone to users.
timer_id_jam_warningTimer id for the looping jam warning timer
